Dashboard-Siniestralidad-Vial. README.

DESCRIPCION:

Este proyecto cuenta con una barra de navegación(introducción, tabla y dashboard) donde podrás explorar la pagina web. Existe una matriz(o tabla) la cual esta conectada con el dashboard, podrás descargarla en un archivo.pdf(no actualizable pero estética) o en un archivo.xlsx(actualizable pero no estetico). El proyecto cuenta con un dashboard estético y lineal con el diseño de la pagina web, este dashboard muestra diferentes graficas(cada una con información diferente) y kpi. Si los datos de la tabla son modificados desde el archivo.html, el dashboard también sufrirá un cambio.

Este proyecto es una simulación de la solución a la problemática de "información rápida a partir de unos datos" para una empresa de seguridad vial(FINSTRUVIAL). Todos los derechos de imagen le pertenece a esta empresa REAL.

INSTALACION:

Descarga el archivo.zip "Dashboard de siniestralidad vial.zip". Ingresas a la carpeta extraída, buscas la carpeta "Código Principal" y accedes al archivo "dashboard_intro.html" para poder empezar a navegar sobre este proyecto.
